Sensors on a production line

The industrial sensor families in one working tour — RTD/thermocouple, pressure transmitters, flow and level, proximity and photoelectric — each with its signal, failure modes, and wiring. The 4–20 mA loop derived as an engineering decision, live zero included. A filling line's sensor set walked end to end.
