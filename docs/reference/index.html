RESOURCESTools, downloads, and study material

Resources

Each course's own equations, notation, and key terms now live on that course's own Reference tab, right alongside its syllabus. This page compiles the two things that genuinely span courses: the software this curriculum actually uses, and printable summaries at whatever scope you need — one course, one semester, or a whole year.

Tools & software directory

CAD

AutoCAD

Autodesk

2D/3D drafting standard; free for verified students via Autodesk's education plan.

Used in: DRW 102

Fusion 360

Autodesk

CAD, CAM, and basic simulation in one package; free education license.

Used in: DRW 102, MFG 154

SolidWorks

Dassault Systèmes

Parametric 3D mechanical design; the CSWA/CSWP certifications are recognized industry-wide.

Used in: DRW 102, MFG 105, MFG 154

CAM & Manufacturing

Mastercam

CNC Software

The most widely installed CAM package for generating CNC toolpaths from a 3D model.

Used in: MFG 154

Circuit Simulation

LTspice

Analog Devices

Free SPICE circuit simulator — the de facto standard for student and professional circuit analysis alike.

Used in: ELE 156, ELX 205

FEA & Simulation

ANSYS Fluent

ANSYS

The standard CFD package for fluid flow and convective/conductive heat-transfer simulation.

Used in: HTX 253, FLD 203

ANSYS Mechanical

ANSYS

The standard structural finite-element analysis package — stress, fatigue, and vibration simulation.

Used in: STA 103, SOM 201, MAT 153, MFG 105

Materials Selection

Granta EduPack

ANSYS (Granta)

The standard materials-selection and Ashby-chart teaching tool used across materials science courses.

Used in: MAT 104, MAT 153

Numerical Computing

MATLAB

MathWorks

Free student-priced license at most universities; the standard numerical/matrix computing environment for engineering coursework.

Used in: MTH 101, MTH 151, MTH 207, PHY 107, STA 103, DYN 152, SOM 201, CMP 106, THM 155, THM 202, HTX 253, FLD 203

Python (NumPy / SciPy / Jupyter)

Python Software Foundation

Free and open source; NumPy/SciPy/Matplotlib cover the same numerical ground as MATLAB at zero license cost.

Used in: MTH 101, MTH 151, MTH 207, PHY 107, DYN 152, CMP 106, STS 206

Simulink

MathWorks

Block-diagram simulation on top of MATLAB — circuits, controls, and dynamic systems without writing code.

Used in: ELE 156, ELX 205

Statistics

Minitab

Minitab

The standard statistical software in industrial quality and Six Sigma programs.

Used in: STS 206

Thermal Systems

EES (Engineering Equation Solver)

F-Chart Software

Built-in thermodynamic and transport property tables — the standard tool for cycle and heat-transfer coursework.

Used in: THM 155, THM 202, HTX 253, FLD 203
